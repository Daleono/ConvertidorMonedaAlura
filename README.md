# Convertidor Monetario Alura

![Badge en Desarollo](https://img.shields.io/badge/STATUS-EN%20DESAROLLO-green)

## Descripción
Proyecto desarrollado como ejercicio del manejo de APIs. El conversor de monedas ofrece calcular el valor en dolares(USD) a otras denominaciones, tales como: peso Argentino(ARS), real brasileño(BRL) y peso colombiano(COP).

La información es obtenida por medio de ExchangeRate-API. Aplicación de consola, el usuario podrá navegar a través de nuestra sencilla interfaz.
